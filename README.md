# Hestia - An Apartment Manager

As mentioned in the description of this repo, the intention behind this project is to create a (sort of) demo version of an apartment manager. This is a hobby project. The name "Hestia" was inspired by the Greek goddess "Hestia" of the hearth, home, and family. And yes, I went with the corny name "Hestia", deal with it.

More info: [Hestia on Wikipedia](https://en.wikipedia.org/wiki/Hestia)
